# 🏥 Medical AI Assistant System

Complete cardiovascular risk assessment system combining Machine Learning and RAG (Retrieval-Augmented Generation).

## 🎯 Features

- ❤️ **Heart Disease ML Prediction** (Logistic Regression, 85.7% accuracy)
- 💬 **Medical RAG Chatbot** (LangChain + Ollama + ChromaDB)
- 🔄 **Combined Analysis** (ML + RAG integrated)
- 📊 **Professional Dashboard** (Streamlit UI)
- 🔧 **Technical Documentation** (For academic proposals)
